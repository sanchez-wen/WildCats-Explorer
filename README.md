WildCats Explorer

Aplicación web interactiva desarrollada como proyecto final de la materia, que permite explorar información detallada sobre felinos salvajes y consultar datos dinámicos mediante una API externa.

Descripción del Proyecto
WildCats Explorer es una aplicación web que combina:

- Información estática de 12 felinos salvajes
- Consumo de API externa (TheCatAPI)
- Interacción dinámica con JavaScript
- Diseño responsivo
  
El usuario puede:
- Explorar fichas técnicas completas
- Visualizar galerías de imágenes
- Buscar felinos por nombre
- Cambiar entre felinos locales y gatos obtenidos desde API
- Visualizar información detallada en un modal dinámico

Tecnologías Utilizadas

- HTML5 (estructura semántica)
- CSS3 (diseño responsivo y animaciones)
- JavaScript (manipulación del DOM y eventos)
- Fetch API (consumo de datos externos)
- TheCatAPI (API pública de gatos)
- GitHub (control de versiones)
- GitHub Pages / Netlify (despliegue en la nube)

Funcionalidades

Felinos Salvajes
- 12 especies con ficha técnica completa
- Información científica detallada
- Galería adicional de imágenes
- Modal dinámico con datos extendidos
- Buscador en tiempo real

Gatos API
- Consumo de API externa usando fetch
- Visualización de:
  - Imagen
  - Nombre de la raza
  - Origen
  - Temperamento
  - Esperanza de vida

Interactividad
- Animaciones suaves
- Transiciones dinámicas
- Contador automático de resultados
- Diseño adaptable a dispositivos móviles
